#### 目录结构

- config存放默认文件
- lib存放操作数据库文件
- middlewares存放中间件
- public存放样式文件
- views存放模板文件
- index程序主文件



#### 安装的模块
1. `koa ndoe` 框架
2. `koa-bodyparser` 表单解析中间件
3. `koa-mysql-session`、`koa-session-minimal`处理数据库的中间件
4. `koa-router` 路由中间件
5. `koa-static`静态资源加载中间件
6. `ejs` 模板引擎
7. `md5`密码加密
8. `moment`时间中间件
9. `mysql`数据库
10. `koa-views`模板呈现中间件
